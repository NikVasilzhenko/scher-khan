Надежная защита

Радиоканал брелока защищен с помощью алгоритма AES-128 — одного из самых криптостойких в мире.

Используется помехоустойчивая частота 868 МГц, что гарантирует надежную радиосвязь в городских условиях

При связи по Bluetooth используются защищенные режимы с динамическими ключами.

Ключ шифрования уникален для каждого брелока

Режим шлюза в системе Long Link: связь смартфона с брелоком происходит по Bluetooth.
